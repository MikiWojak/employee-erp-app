# frontend-vue

## Project setup

```bash
# Set proper node version
nvm use

# Install dependencies
npm install

# Copy env file
cp .env.example .env
```

### Compiles and hot-reloads for development

```
npm run dev
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
